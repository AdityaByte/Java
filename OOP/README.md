## OOPS - Object Oriented Programming

Class - A Class is a named group of properties and things <br>
<ol>
<li>If we want to create our own datatype we need classes and it comes under the catergory of non-primitive data types and it can act as a template also</li>
<li>A Class is a template of an object and object is an instance of a class</li>
</ol>
Means - Instance means it justs a physical stuff like when babies are born they are the instance of the human class

Class - Logical construct
Object - Physical reality // occupying space in the memory

Object - We can say that object is an instance of a class which occupies some space in the primary memory <br>
Object has the following three properties - <br>
<ul>
  <li>State of the object</li>
  <li>Identity of the object</li>
  <li>Behaviour of the object</li>
</ul>

 <span>NOTE - Instance Variable - variable inside the objects are called the instance variable </span><br>
<span>NOTE - Dot operator - In java is was termed as seperator and it is used for linking refernce variable to the instance variable </span><br>
<span>NOTE - New keyword - It allocates the memory at runtime ( dynamically allocates ) and return a reference to it</span><br>
<span>NOTE - Objects are created at heap memory and all class objects in java must be allocated dynamically</span><br>
<span>NOTE - We can't access the memory address because in java it's not allowed so this is a key for memory safety as we can't manipulate the references as you can pointer in C++</span><br>
