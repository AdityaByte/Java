## OOPS - Object Oriented Programming

Class - A Class is a named group of properties and things <br>
1) If we want to create our own datatype we need classes and it comes under the catergory of non-primitive data types and it can act as a template also <br>
2) A Class is a template of an object and object is an instance of a class <br>
Means - Instance means it justs a physical stuff like when babies are born they are the instance of the human class

Class - Logical construct
Object - Physical reality // occupying space in the memory

Object - We can say that object is an instance of a class which occupies some space in the primary memory <br>
Object has the following three properties - <br>
i) State of the object <br>
ii) Identity of the object <br>
iii) Behaviour of the object <br>
