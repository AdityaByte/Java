
# Web Scarpping Using Java

Web scraping, or web crawling, refers to the process of fetching and extracting arbitrary data from a website. This involves downloading the site's HTML code, parsing that HTML code, and extracting the desired data from it



## JSOUP

Jsoup is a java library or dependency for web scrapping or in general terms we can say that it is the library used for retreiving data from the html web page like title elements info etc..




## Documentation

[JSOUP Official Documentation](https://jsoup.org/)


## Maven Depolyment

To use the jsoup library you have to add it in the dependencies of your project for this follow the following steps and the following steps are for the intellij idea ide...

##### 1. Make a new project named anything you want and make it using maven.
##### 2. Inside the pom.xml file add this code -

```bash
  <dependency>
  <!-- jsoup HTML parser library @ https://jsoup.org/ -->
  <groupId>org.jsoup</groupId>
  <artifactId>jsoup</artifactId>
  <version>1.17.2</version>
  </dependency>
```

##### 3. After that restart the mavan and write you code inside the src/main.java file
