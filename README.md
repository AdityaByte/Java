# Java Repository

Welcome to the **Java** repository! This repository contains a collection of Java programs, including both core Java and advanced Java applications. The projects showcase various Java programming techniques and technologies.

## Overview

This repository is organized into two main categories:

1. **Core Java Programs**: Basic Java programs that demonstrate fundamental programming concepts, including object-oriented principles, data structures, algorithms, and more.

2. **Advanced Java Programs**: More complex Java applications that utilize advanced features and frameworks such as Java Servlets, Spring Framework, Hibernate ORM, and more.

## Project Structure

- **core-java/**: Contains programs and exercises focused on core Java concepts.
- **advanced-java/**: Contains programs and applications that leverage advanced Java technologies and frameworks.
